# Friendly Pikes Helper

#### A basic and rather silly bot made for friendly pikes

## Features
- Mostly modulator!

- Messages
  - Logs (Delete and Edit)
  - owo reaction (new messages)

- Commands
  - Users
    - cutedar
    - sillydar
    - bidar
    - gaydar
    - queerdar
    - rizzdar
    - transdar

  - Staff
    - ban (Diffrent from banish)
    - kick
    - banish
    - unbanish
    - reply
    - repeat

  - Staff Silly
    - explode (explodeinator)
    - cutinate (cuteinator)
    - smolinate (smolinator)
    - tallinate (tallinator, recommened by Pixie - the owner)
    - sillinate (sillyinator, recommended by Koodoger)
    - unexplode (explodeinator)
    - uncutinate (cuteinator)
    - unsmolinate (smolinator)
    - untallinate (tallinator, recommened by Pixie - the owner)
    - sillinate (sillyinator, recommended by Koodoger)
